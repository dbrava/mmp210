MMP 210

COURSE DESCRIPTION

This course introduces the basic concepts of programming for multimedia.
Students will learn the principles of object oriented programming and how to create scripts for the manipulation of video, graphics, and text to construct a complete multimedia presentation.

Assignments

  • 1 - Self-portrait
  • 2 - Drawing with Variables
  • 3 - Logo
  • 4 - System Variables
  • 5 - Windows of New York

Tips and Tricks

Command+Slash is used to effectively find bugs and/or modify without deleting the original. Select the part you want and press command/slash
